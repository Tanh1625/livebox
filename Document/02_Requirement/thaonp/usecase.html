1.3 User Requirements

1.3.1 Actors

Based on the Project Context, the LiveBox Real-time Chat Platform interacts directly with three primary System Actors:

No. Actor Description & System Expectations
1 Server Owner (P01 - Primary User) Description: The creator and administrator of a Server. They have full control over the Server's configuration, channels, and members.
Responsibilities: Create Server, create/delete channels, generate invite links, and perform kick/ban moderation actions.
2 Active Member (P02 - Core User) Description: A regular user who has joined one or more Servers. They are the primary participants in the real-time communication platform.
Responsibilities: Send and receive real-time text messages, participate in WebRTC voice channels (max 20 users), and update their display name/avatar.
3 Guest User (P03 - Casual User) Description: A user who has received an invite link but has not yet registered, or a newly registered user who is just joining a Server for the first time.
Responsibilities: Register an account (in under 30 seconds), log in, and join their first Server using an invite code.
